# Taalwiz Web

The primary Taalwiz client: an Angular 20 + Ionic 8 hybrid web/mobile app, packaged for native platforms with Capacitor 7. It provides the learner-facing experience — content articles, an offline-first dictionary, vocabulary lists, SRS flashcards, and an admin area — talking to the Taalwiz API over a JWT-secured REST interface.
